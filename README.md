# XML Editor RW
---
Version 0.0.2

An XML Generation tool for Rimworld 1.3 written in java 8 (jre 1.8)
 
---
FEATURES:

-An XML template generator

-Generate templates for: ThingsDefs (ThingDef_Item_Base, ThingDefs_Item_Generic, etc).

-choose default or no values
